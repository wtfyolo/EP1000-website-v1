CSS

CSS is used for editing website layouts and styles like fonts, positioning of text , colours and can be used internally or externally

for css each of the section has a declare a object { insert code here } and parathesis that is the body of the data is where the code in


color: (insert color) ; is used to declare text color background color: (insert color); is used to declare what what background color is used

next is fonts font-family :(family name); is usedto declare font family types like helvetica font-weight : (value); is used to declare thickness of the font; font-size : (value); is used to declare size of font usually in the form of px or wordsize like LARGE;

positioning there are multiple ways to position text items and images personally i use position : relative; change position relative to original location and left : (value); move how much left by pixels , percentages relative to the body size top : (value); move how much from the top relative to size

computer graphics and computer aided drafting

for 2d graphics we used 2 different software raster graphics and vector graphics

Gimp

the first being raster graphics which allows you to edit raster files which are pixel based arranged in a matrix to be form images files include in the form of Jpeg , Png, GIF. for the exercise we used GIMP (GNU Image Manipulation Program) to edit the image by using the free select tool then adding a mask layer and turning it invisible to get a image of just the object we want.

Inkscape

next was vector graphics which uses mathmatical equations to draw out graphics the resultant image is often scaleable without losing quality or looking pixelated unlike raster based graphics files include SVG,Pdf,EPS,Dvg.

Fusion 360

for 3d cad drafting we used Fusion 360 a 3d cad software that uses parametric modelling techniques defined by user dimensions. we can model in 2d to create 2d drawings or 3d drawing objects. it uses the form of sketches to create a basic sketch that can be extruded using the extrusion tool.
